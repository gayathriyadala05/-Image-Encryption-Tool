# Image Encryption Tool using Pixel Manipulation

This project is developed as part of my Cyber Security Internship at SkillCraft Technology.

The program encrypts an image by modifying pixel values using a simple XOR operation in Java. 
It reads an input image, manipulates RGB values of each pixel, and generates an encrypted image.

## Technologies Used
- Java
- BufferedImage
- ImageIO
